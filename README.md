# LiveProxySpeedTest
🌩「自选优选 IP」测试直播代理 IP 延迟和速度，获取最快 IP (IPv4 / IPv6)！暂时支持百视通BesTV、爱尚AiShang

### 使用
在 [Release](https://github.com/sec-an/LiveProxySpeedTest/releases) 下载解压后运行
```shell
LiveProxySpeedTest -t bestv # 百视通
LiveProxySpeedTest -t aishang # 爱尚
```
